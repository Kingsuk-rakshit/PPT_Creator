# PPT Creator

An AI-powered PowerPoint generator built with **Streamlit**, **Groq LLM**, and **python-pptx**.  
It creates complete presentation slides — including titles, bullet points, and relevant images — based on a topic you provide through a chat interface.

## Features
- Automatically generates structured slide content  
- Allows feedback-based refinement  
- Fetches images from Pexels  
- Builds downloadable `.pptx` files  
- Supports theme color customization  

## Usage
Enter a topic → review the generated outline → confirm → download your fully prepared PowerPoint.
